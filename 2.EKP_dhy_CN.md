## 酶动力学参数预测项目

### 介绍
- **背景**：预测酶动力学参数是酶工程、代谢建模和生物催化剂发现的中心问题。在实践中，实验测量周转数（$k_{\text{cat}}$）、米氏常数（$K_m$）和抑制常数（$K_i$）成本高且耗时。最近的研究如 **CatPred** 和 **ERBA** 表明，多模态学习可以结合蛋白质语言模型、底物编码和结构信息来改进预测精度和样本外推广性能。这些工作基于从 **BRENDA** 和 **SABIO-RK** 衍生的精选基准数据集，使酶动力学预测成为一个具有公开文档化数据集、评估设置和可复现基线的现实机器学习项目。

- **学习目标**：
  - 掌握将酶动力学预测制定为多模态回归问题的方法
  - 理解如何整合蛋白质序列、底物化学和结构上下文
  - 学习如何从生化数据库中精选基准数据集
  - 比较仅序列、序列-底物和结构感知学习方法

### 问题陈述
- **技术描述**：开发机器学习方法来预测给定酶-底物对的酶动力学参数，可选择使用活性位点或全蛋白质结构信息。
- **数学定义**：
  - **阶段 1：酶-底物表征学习**
    - 输入：
      - 酶序列：$S_e \in \Sigma^n$
      - 底物分子：$S_m$
      - 可选结构/上下文：$S_g$
    - 编码输出：
      - $H_e \in \mathbb{R}^{n \times d}$：酶嵌入
      - $H_m \in \mathbb{R}^{m \times d}$：底物嵌入
      - $H_g \in \mathbb{R}^{g \times d}$：几何/结构嵌入
    - 学习任务：多模态特征对齐和融合
    - 示例目标函数：
      - $\min_{\theta} \mathcal{L}_{\text{repr}} = \mathcal{L}_{\text{align}} + \lambda_1 \mathcal{L}_{\text{fusion}}$

  - **阶段 2：动力学参数回归**
    - 输入：融合表征 $z = f(H_e, H_m, H_g)$
    - 输出：
      - $\hat{y} \in \mathbb{R}^3 = [\log_{10}(k_{\text{cat}}), \log_{10}(K_m), \log_{10}(K_i)]$
      - 或单个任务的单端点回归目标
    - 学习任务：具有可选异方差不确定性估计的回归
    - 示例目标函数：
      - $\min_{\phi} \mathcal{L}_{\text{pred}} = \mathcal{L}_{\text{reg}}(y, \hat{y}) + \lambda_2 \mathcal{L}_{\text{uncertainty}}$
      - 其中 $\mathcal{L}_{\text{reg}}$ 可以是 MSE / MAE / 高斯 NLL
- **关键挑战**：
  - 跨序列、底物和结构的多源数据整合
  - 稀疏和噪声实验测量
  - 跨酶家族和底物的分布转移
  - 某些蛋白质结构覆盖不足
  - 在序列分离或样本外分割下的公平基准测试
- **预期成果**：
  - 一个可复现的酶动力学预测管道
  - 跨序列仅有和多模态模型的基准比较
  - 了解何时结构信息改进预测
  - 一个用于优化酶-底物测定的可部署推理工作流

### 数据描述
- **提供的/核心数据集**：
  - **CatPred 基准数据集**：
    - $k_{\text{cat}}$：约 23k 个数据点
    - $K_m$：约 41k 个数据点
    - $K_i$：约 12k 个数据点
  - **ERBA 主数据集**：
    - 基于从先前工作衍生的精选动力学端点数据集，最终来自 **BRENDA** 和 **SABIO-RK**
  - **样本外评估数据集**：
    - 基于 EITLEM 的 $k_{\text{cat}}$ 和 $K_m$ 测试集用于样本外评估
- **额外数据源**：
  - **BRENDA**：实验测量的酶动力学记录
  - **SABIO-RK**：精选的反应动力学条目
  - **UniProt**：已验证的酶序列和标识符
  - **PubChem / KEGG / RDKit 衍生的规范 SMILES**：底物标准化
  - **AlphaFold / ESMFold / OpenFold** 结构：用于缺乏实验结构的蛋白质
- **变量**：
  - 酶氨基酸序列
  - 底物名称/规范 SMILES/分子图
  - 动力学端点（$k_{\text{cat}}$、$K_m$、$K_i$）
  - 酶类（EC 编号）
  - 生物体元数据
  - 可选的 3D 结构/活性位点口袋描述符
- **样本量**：
  - 主基准规模约为 `23k + 41k + 12k` 个标记示例，跨三个端点
  - 样本外集包括数万个在更严格过滤下收集的 $k_{\text{cat}}$ 和 $K_m$ 条目
- **数据集可用性**：`https://github.com/maranasgroup/CatPred-DB`

### 方法论
- **建议的方法**：
  - 用于序列编码的蛋白质语言模型（ESM2、ProtT5、Ankh）
  - 用于底物编码的分子图神经网络/指纹
  - 用于口袋几何或残基图的结构感知编码器
  - 交叉注意或分阶段多模态融合（序列 → 底物 → 结构）
  - 不确定性感知回归头（异方差高斯 NLL）
  - 跨 $k_{\text{cat}}$、$K_m$ 和 $K_i$ 的多任务学习
- **实施指南**：
  - 在可用时使用规范的训练/验证/测试分割
  - 报告序列身份感知或样本外性能，而不仅仅是随机分割
  - 至少比较三种模型设置：
    - 仅序列
    - 序列 + 底物
    - 序列 + 底物 + 结构
  - 文档化所有预处理决策：
    - UniProt 映射
    - SMILES 规范化
    - 重复聚合
    - 缺失结构处理
  - 在可能的地方包括不确定性校准或置信度排名

### 评估
- **建议的指标**：
  - 回归性能：
    - $R^2$
    - 皮尔逊相关系数（PCC）
    - 平均绝对误差（MAE）
    - 均方根误差（RMSE）
  - 鲁棒性/推广性：
    - 不同酶样本外性能
    - 不确定性估计下的校准
    - EC 类别细分
- **验证**：
  - 固定训练/验证/测试分割评估
  - 如果构建缩减的教学子集，则进行五折交叉验证
  - 独立样本外测试集评估
  - 结构使用和融合顺序的消融研究
